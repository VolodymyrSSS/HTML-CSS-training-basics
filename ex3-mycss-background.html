CSS Backgrounds

CSS background-color

The background-color property specifies the background color of an element.

The background color of a page is set like this:

style>
body {
background-color: tomato;
}
/style>

With CSS, a color is most often specified by:
a valid color name - like "red"
a HEX value - like "#ff0000"
an RGB value - like "rgb(255,0,0)"
Look at CSS Color Values for a complete list of possible color values.

CSS background-position

The background-position property is used to specify the position of the background image.

body {
background-image: url("./img/owl_tree.jpeg");
background-repeat: no-repeat;
background-position: right top;
margin-right: 200px;
}

Hello World!

Now the background image is only shown once, and positioned away from the text.

In this example we have also added a margin on the right side, so the background image will never disturb the text.

CSS background-attachment

The background-attachment property specifies whether the background image should scroll or be fixed (will not scroll with the rest of the page):

Specify that the background image should scroll with the rest of the page:

background-attachment: scroll;

body {
background-image: url("./img/global-computer-network1.jpg");
background-repeat: no-repeat;
background-position: right top;
margin-right: 20px;
background-attachment: scroll;
}

CSS background - Shorthand property

To shorten the code, it is also possible to specify all the background properties in one single property.
This is called a shorthand property.

The background property is a shorthand property for specifying all the background properties in one declaration.
The shorthand property for background is background.

body {
background: #ADD8E6 url("./img/owl_tree.jpeg") no-repeat right bottom; margin-right: 200px; }

We could add a right margin, so that the text will not write over the background image, but here it is not necessary.

Here, the background image is only shown once, and it is also positioned in the bottom-right corner.

When using the shorthand property the order of the property values is:
- background-color in the example - #ADD8E6
- background-image in the example - url("./img/owl_tree.jpeg")
- background-repeat in the example - no-repeat
- background-attachment in the example - is missing, that means set to scroll
- background-position in the example - right bottom
It does not matter if one of the property values is missing, as long as the other ones are in this order.

CSS Multiple Backgrounds

CSS allows you to add multiple background images for an element, through the background-image property.

The different background images are separated by commas, and the images are stacked on top of each other, where the first image is closest to the viewer.

Multiple Backgrounds

The following example has two background images, the first image is a flower (aligned to the bottom and right) and the second image is a paper background (aligned to the top-left corner):

background-image: url(./img/img_flwr.gif), url(./img/papier.jpg);
background-position: right bottom, left top;
background-repeat: no-repeat, repeat;
padding: 15px;
}

Now we want to have a background image on a website that covers the entire browser window at all times.
The requirements are as follows:

Fill the entire page with the image (no white space)
Scale image as needed
Center image on page
Do not cause scrollbars

The following example shows how to do it;
Use the html-element (the html-element is always at least the height of the browser window).
Then set a fixed and centered background on it.
Then adjust its size with the background-size property.

Hero Image

You could also use different background properties on a div to create a hero image (a large image with text), and place it where you want.

.hero-image {
background: url(img_man.jpg) no-repeat center;
background-size: cover;
height: 500px;
position: relative;
}

CSS background-origin Property

The CSS background-origin property specifies where the background image is positioned.

The property takes three different values:
border-box - the background image starts from the upper left corner of the border
padding-box - (default) the background image starts from the upper left corner of the padding edge
content-box - the background image starts from the upper left corner of the content
