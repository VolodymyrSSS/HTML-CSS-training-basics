CSS 3D Transforms

CSS also supports 3D transformations

CSS 3D Transforms Methods

    With the CSS transform property you can use the following 3D transformation methods:
  1. rotateX()
  2. rotateY()
  3. rotateZ()

Some older browsers need specific prefixes (-webkit-) to understand the 3D transform properties

The rotateX() Method

The rotateX() method rotates an element around its X-axis at a given degree
Note: Internet Explorer 9 (and earlier versions) does not support the rotateX() method.

This div element is rotated 150 degrees

The code for above div-element:
div {
-webkit-transform: rotateX(150deg); /* Safari prior 9.0 */
transform: rotateX(150deg); /* Standard syntax */
}
