This example demonstrates how to use the :focus pseudo-class:

The result will be visible only to those elements that can get focus. In particular, these are the tags 'a', 'input', 'select' and 'textarea'.

This example demonstrates how to use the :focus pseudo-class;
As soon as you click into input-field it changes it's back-ground color to yellow

The code for this:
input:focus {
background-color: yellow;
}

This example demonstrates how to use the :visited pseudo-class:

This pseudo-class selects all visited links.
Typically, such a link changes its default color to purple, but with the help of styles, color and other parameters can be set independently.

The code for this:
a:visited {
color: blue; - when we want to change the default color
}

We can use the colors for the link 'a' like this:
a:link { color: red } /* unvisited links */
a:visited { color: blue } /* visited links */
a:hover { color: yellow } /* user hovers */
a:active { color: lime } /* active links */

This example demonstrates how to use the :last-child pseudo-class:

Selects every elements that is the last child of its parent.

There are two bolded elements in every parent p-element

This pseudo-class selects every bolded elements that is the last child of its parent p-element
and add a background color.

The code for this:
b:last-child {
background: lightsalmon;
}

This example demonstrates how to use the :nth-child(n) pseudo-class:

Selects every element that is the n-th child of its parent.
It's mean that you can calculate specific child target element.

If you want to aim at the first point and every third point after it, then in the styles you need to write:

li.proverbs:nth-child(3n+1) {
background: yellowgreen;
}
