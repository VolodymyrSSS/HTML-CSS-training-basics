Додати рамку з правої сторони, не змінюючи ширину елементів

Є 2 div-елементи які представляють дві колонки шириною 30% та 70% відповідно;
- Потрібно додати тільки до правої колони зелену рамку (border) шириною 2рх та зробити відступ для контенту з лівої сторони (padding-left); при цьому, двоколонна верстка не повинна зламатись;
- Не можна змінювати властивість width ні зправа ні зліва та не створювати додаткових елементів;

Для вирішення цієї задачі потрібно задати для другої колонки зелену рамку шириною 2рх. А для збереження ширини, після того як буде додана рамка, надати цій колонці властивість box-sizing: border-box; та відповідний префікс для універсальності. Відступ для контексту робимо звичайним padding-left. Ось так:

.right2 {
float: right;
width: 70%;
box-sizing: border-box;
-moz-box-sizing: border-box;
border: 2px solid green;
padding-left: 20px;
background: tan;
}
