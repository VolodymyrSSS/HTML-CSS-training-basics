CSS Gradients

CSS-градиент представляет собой переходы от одного цвета к другому.
CSS gradients let you display smooth transitions between two or more specified colors.

CSS defines two types of gradients:

- Linear Gradients (using the keywords: to top, to right, to bottom, to left, to bottom right - diagonally or using the angles that match keywords - a gradient angle equal to 0deg, 90deg, 180deg and 270deg respectively)
- Radial Gradients (defined by their center)

Градиенты создаются с помощью функций linear-gradient() и radial-gradient().

Gradient background can be set in the properties:
background, background-image, border-image и list-style-image.

CSS Linear Gradients

To create a linear gradient you must define at least two color stops.
Color stops are the colors you want to render smooth transitions among.
You can also set a starting point and a direction (or an angle) along with the gradient effect.

Syntax
background-image: linear-gradient(direction, color-start, color-end);

By default, gradient colors are evenly distributed in the direction perpendicular to the gradient line.

Linear Gradient - Top to Bottom (this is default)
The following examples shows a linear gradient that starts at the top. It starts red, transitioning to yellow:

background-image: linear-gradient(red, yellow);

Linear Gradient - Left to Right
The following example shows a linear gradient that starts from the left. It starts red, transitioning to yellow:

background-image: linear-gradient(to right, red , yellow);

Linear Gradient - Diagonal
You can make a gradient diagonally by specifying both the horizontal and vertical starting positions.
The following example shows a linear gradient that starts at top left (and goes to bottom right). It starts red, transitioning to yellow:

background-image: linear-gradient(to bottom right, red, yellow);

Linear Gradient - Left to Right another symple using HEX for colour

background: linear-gradient(to right, #F6EFD2, #CEAD78);

If you want more control over the direction of the gradient, you can define an angle, instead of the predefined directions.
The angle is specified as an angle between a horizontal line and the gradient line.
The following example shows how to use angles on linear gradients:

0deg

90deg

180deg

-90deg

Linear Gradient - use angle another symple has taken HEX for colours

background: linear-gradient(45deg, #EECFBA, #C5DDE8);

Using Multiple Color Stops

Syntax
background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

The following example shows a linear gradient (from top to bottom) with multiple color stops:


3 Color Stops, evenly spaced:


background-image: linear-gradient(red, yellow, green);

7 Color Stops, evenly spaced:

background-image: linear-gradient(red, orange, yellow, green, blue, indigo, violet);

3 Color Stops, not evenly spaced:

background-image: linear-gradient(red 10%, green 85%, blue 90%);

Using Transparency

CSS gradients also support transparency, which can be used to create fading effects.

To add transparency, we use the rgba() function to define the color stops.
The last parameter in the rgba() function can be a value from 0 to 1, and it defines the transparency of the color: 0 indicates full transparency, 1 indicates full color (no transparency).

The following example shows a linear gradient evenly spaced that starts from the left.
It starts fully transparent, transitioning to full color red:

background-image: linear-gradient(to right, rgba(255, 0, 0, 0), rgba(255, 0, 0, 1));

Repeating a linear-gradient

The repeating-linear-gradient() function is used to repeat linear gradients.


A repeating gradient from top to bottom starting red and finishing green:

background-image: repeating-linear-gradient(red, yellow 10%, green 20%);

A repeating gradient on 45deg axe starting red and finishing green:

background-image: repeating-linear-gradient(45deg, red, yellow 7%, green 10%);

A repeating gradient on 190deg axe starting red and finishing green:

background-image: repeating-linear-gradient(190deg, red, yellow 7%, green 10%);

A repeating gradient on 90deg axe starting red and finishing green:

background-image: repeating-linear-gradient(90deg, red, yellow 7%, green 10%);

Other repeating linear-gradient on 45deg axe

background: repeating-linear-gradient(45deg, #606dbc, #606dbc 10px, #465298 10px, #465298 20px);}

CSS Radial Gradients

A radial gradient is defined by its center.
To create a radial gradient you must also define at least two color stops.

Syntax
background-image: radial-gradient(shape size at position, start-color, ..., last-color);

By default, shape is ellipse, size is farthest-corner, and position is center.


Radial Gradient - Evenly Spaced Color Stops (this is default)

background-image: radial-gradient(red, yellow, green);

Radial Gradient - Differently Spaced Color Stops

background-image: radial-gradient(red 5%, yellow 15%, green 60%);

Radial Gradient - Evenly Spaced Color Stops another symple

background: radial-gradient(white, #FFA9A1);

CSS Radial Gradients - Set Shape

The shape parameter defines the shape. It can take the value circle or ellipse. The default value is ellipse.

The following example shows a radial gradient with the shape of ellipse (this is default):

background-image: radial-gradient(red, yellow, green);

The following example shows a radial gradient with the shape of a circle:

background-image: radial-gradient(circle, red, yellow, green);

The center position is set using the keywords used in the background-position property, with the addition of the at prefix.
If no center position is specified, the default value of at center is used.

background: radial-gradient(at top, #FEFFFF, #A7CECC);

Radial Gradients - Different size keywords

The size parameter defines the size of the gradient. It can take four values:
closest-side
farthest-side
closest-corner
farthest-corner

The following examples show the different size keywords:

closest-side

farthest-side

closest-corner

farthest-corner (default)

Other example is for circle farthest-corner positionin:

radial-gradient(circle farthest-corner at 100px 50px, #FBF2EB, #352A3B);

Repeating a radial-gradient

The repeating-radial-gradient() function is used to repeat radial gradients.

Repeating Radial Gradient Differently Spaced Color Stops

background-image: repeating-radial-gradient(red, yellow 10%, green 15%);

Other Repeating Radial Gradient Differently Spaced Color Stops

background: repeating-radial-gradient(circle, #B9ECFE, #B9ECFE 10px, #82DDFF 10px, #82DDFF 20px);
