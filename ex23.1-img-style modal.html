Image Modal (Advanced)

Let's look at the example to demonstrate how CSS and JavaScript can work together.
First, use CSS to create a modal window (dialog box), and hide it by default.
Then, use a JavaScript to show the modal window and to display the image inside the modal, when a user clicks on the image:

Ã—

In this example, we use CSS to style the image itself and to create a Modal (modal window or dialog box) that is hidden by default.

/*Style the image*/
#myImg {
display: block; /* to show placed image in a page */
margin-left: auto; /* position on the middle of the page */
margin-right: auto; /* position on the middle of the page */
border-radius: 5px; /* round the corner of the image */
cursor: pointer; /* change cursor into pointer */
transition: 0.3s; /* time while transit */
}

#myImg:hover {
opacity: 0.7; /* change opacity on hover */
}

.modal { /* the Modal (background) */
display: none; /* hidden by default */
position: fixed; /* stay in place */
z-index: 1; /* sit on top */
padding-top: 100px; /* location of the box */
left: 0; /* location of the box */
top: 0; /* location of the box */
width: 100%; /* full width */
height: 100%; /* full height */
overflow: auto; /* enable scroll if needed */
background-color: rgb(0, 0, 0); /* fallback color */
background-color: rgba(0, 0, 0, 0.9); /* black w/ opacity */
}

We use again CSS to create Modal Content (that is our image) and Caption, after add animation

.modal-content {
display: block; /* to show placed image in a modal */
width: 80%; /* full width of the image im a modal */
max-width: 700px; /* max width of the image im a modal */
margin: auto; /* place image on a middle of the modal*/
}

#caption {
display: block; /* display caption in the modal */
width: 80%; /* full width of the caption in a modal */
max-width: 700px; /* max width of the caption in a modal */
height: 150px; /* full height of the caption */
text-align: center; /* aligne the text of the caption in the modal */
color: #ccc; /* text-color of the caption */
padding: 10px 0; /* position caption in the modal */
margin: auto;/* place caption in a middle of the page */
}

/* Add Animation */
.modal-content,
#caption {
-webkit-animation-name: zoom;
-webkit-animation-duration: 0.6s;
animation-name: zoom;
animation-duration: 0.6s;
}

@-webkit-keyframes zoom {
from {
-webkit-transform: scale(0) /* Safari prior 9.0 */
}

to {
-webkit-transform: scale(1) /* Safari prior 9.0 */
}
}

@keyframes zoom {
from {
transform: scale(0.1)
}

to {
transform: scale(1)
}
}

Then, create like-close button and forsee the use in small-screen devices (makes it responsive)

.close {
position: absolute;
top: 15px;
right: 35px;
color: #f1f1f1;
font-size: 40px;
font-weight: bold;
transition: 0.3s;
}

.close:hover,
.close:focus {
color: #bbb;
text-decoration: none;
cursor: pointer;
}
/* 100% Image Width on Smaller Screens */
@media only screen and (max-width: 700px) {
.modal-content {
width: 100%;
}
}

Only after that, we use JavaScript to trigger the modal and to display the current image inside the modal when it is clicked on.
Also note that we use the value from the image's "alt" attribute as an image caption text inside the modal.

// Get the modal
var modal = document.getElementById('myModal');

// Get the image and insert it inside the modal - use its "alt" text as a caption
var img = document.getElementById('myImg');
var modalImg = document.getElementById("img01");
var captionText = document.getElementById("caption");
img.onclick = function(){
modal.style.display = "block";
modalImg.src = this.src;
captionText.innerHTML = this.alt;
}

// Get the span-element that closes the modal
var span = document.getElementsByClassName("close")[0];

// When the user clicks on 'span' (x), close the modal
span.onclick = function() {
modal.style.display = "none";
}
