CSS Застосування псевдо-елементів (pseudo-elements)

Застосування псевдоелементу ::after

Застосовується для вставки визначеного контенту після основного вмісту елементу.
Цей псевдоелемент працює разом з стилевою властивістю content, яка визначає вміст цієї вставки.

Спосіб зловити лева в пустелі за допомогою методу золотого січення.

Додамо після тексту параґрафа псевдоелемент з текстом, ось так:
p.new::after {
content: " - Новий спосіб!";
}

Застосування псевдоелементу ::before

По свїй дії :before аналогічний псевдоелементу :after, але вставляється контент до вмісту основного елементу.
В якості аргумента не обов'язково може бути тільки текст, тут також можуть бути застосовані символи юнікоду.

Ось маємо простий список:
  • Чебуреки
  • Пиріжки
  • Піца
  • Тортики
Тепер додамо спереду до списку псевдоелемент з контентом символу юнікоду, ось так:
ul.withel-ps li::before {
}
В результаті отримаємо:
  • Чебуреки
  • Пиріжки
  • Піца
  • Тортики

Застосування псевдоелементу ::first-letter

Визначає стиль першого символу елемента в тексті, до якого він додається. Це дозволяє створити в тексті буквицю (збільшена перша буква, базова лінія якої нижче на одну чи декілька рядків базової лінії основного тексту) та виступаючий ініціал (збільшена прописна буква, базова лінія якої співпадає з базовою лінією основного тексту).

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Corrupti facere non ad? Vero, optio?
Necessitatibus ex perferendis eius illo dolorem rem pariatur molestiae quas, distinctio exercitationem.
Distinctio harum in aperiam! Amet consectetur adipisicing elit. Corrupti facere non ad? Vero, optio?
Necessitatibus ex perferendis eius illo dolorem rem pariatur molestiae quas, distinctio exercitationem.

The code for this:
p.text5::first-letter {
float: left;
display: block;
color: brown;
font-family: 'Arial', sans-serif;
font-size: 48px;
margin-top: 15px;
margin-bottom: 18px;
}

Луч фонарика высветил старые скрипучие ступени, по которым не далее как пять минут назад в дом поднялся Паша. Оля осторожно приоткрыла дверь и посветила внутрь дома. Луч света, словно нехотя, пробивался сквозь тугую завесу из мрака и пыли.

Взгляд Оли опустился на пол, и она вскрикнула. В пустом помещении никого не было, и лишь на полу валялась порванная туфля Паши.

The code for these:
p.text1::first-letter {
font-family: 'Times New Roman', Times, serif; - шрифт першого символу
font-size: 200%; - роззмір шрифта першого символу
color: red; - колір першого символу
}

Застосування псевдоелементу ::first-line

Визначає стиль першого рядка блочного тексту. Довжина цього рядка залежить від багатьох факторів, таких як шрифт, що використовується, розмір вікна браузера, ширина блока, мова і т.д..

Интересно, а существует ли способ действительно практичного применения свойства first-line?
Нет, не такого, чтобы можно было бы показать, что это возможно, а чтобы воистину захватило дух от красоты решения, загорелись глаза от скрытых перспектив, после чего остается только сказать себе, что вот это вот, это самое сделать по-другому, также изящно и эффектно просто невозможно.

The code for this:
p.text2:first-line {
color: red; - колір тексту першого рядка
font-style: italic; - шрифт першого рядка: курсив
}

Застосування псевдоелементів ::before і ::after

К псевдоэлементом можно применять такие же стили, как и к «реальным»: изменение цвета, добавление фона, регулировка размера шрифта, выравнивание текста и т.д. Созданные элементы по умолчанию inline-элементы, поэтому при указании высоты или ширины необходимо установить display: block.

Также внутри псевдоэлемента можно использовать картинку вместо обычного текста, и даже добавлять фоновое изображение.

Lorem, ipsum dolor sit amet consectetur adipisicing elit. In commodi nam enim odit tempora?
Cupiditate, nobis. Exercitationem, ratione voluptatibus, numquam, asperiores eum sapiente iusto laborum in vero
excepturi commodi architecto.Qui quos autem sed quaerat accusantium quo inventore ducimus unde accusamus,
dignissimos nostrum amet vero ipsam ea tempore error culpa non dolores pariatur quis officia distinctio corporis.
Nulla, similique quod?

The code for this:
.demo-line::before,
.demo-line::after {
content: "";
display: block;
width: 100%;
height: 2px;
background: green;
margin-top: 5px;
margin-bottom: 6px;
}
