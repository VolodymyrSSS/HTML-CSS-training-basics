CSS Layout - Vertical Align

Center Vertically - Using padding

Simple solution is to use top and bottom padding

In this example, we use the padding property to center the div element vertically:

div {
padding: 50px 0;
border: 3px solid red;
}

I am vertically centered.

In this example, we use padding and text-align to center the div element vertically and horizontally:

.div {
text-align: center;
padding: 70px 0;
border: 3px solid red;
}

I am vertically and horizontally centered.

Center Vertically - Using line-height

Another trick is to use the line-height property with a value that is equal to the height property.

In this example, we use the line-height property with a value that is equal to the height property and text align to center the div element as vertically as horizontally:

.div {
line-height: 200px;
height: 200px;
border: 3px solid green;
text-align: center;
}

I am vertically and horizontally centered using line-height

If the text has multiple lines, add the following:

.div p {
line-height: 1.5;
display: inline-block;
vertical-align: middle;
}

I am vertically and horizontally centered taking in consideration two rows

Center Vertically - Using position & transform

If padding and line-height are not options, a third solution is to use positioning and the transform property

In this example, we use positioning and the transform property to vertically and horizontally center the div element:

.div {
height: 200px;
position: relative;
border: 3px solid green;
}

.div p {
margin: 0;
position: absolute;
top: 50%;
left: 50%;
-ms-transform: translate(-50%, -50%);
transform: translate(-50%, -50%);
}

I am vertically and horizontally centered using position & transform

Note: The transform property is not supported in IE8 and earlier versions.
