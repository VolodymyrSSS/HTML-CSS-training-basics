CSS Rounded Corners

CSS border-radius Property

The CSS border-radius property defines the radius of an element's corners.

Tip: This property allows you to add rounded corners to elements!

Here are three examples:

Rounded corners for an element with a specified background color:

Rounded corners!
The code for this:
border-radius: 25px;
background: #73AD21;
padding: 20px;
width: 200px;
height: 150px;

Rounded corners for an element with a border:

Rounded corners!
The code for this:
border-radius: 25px;
border: 2px solid #73AD21;
padding: 20px;
width: 220px;
height: 150px;

Rounded corners for an element with a background image:

Rounded corners!
border-radius: 25px;
background: url(./img/paper.gif);
background-position: left top;
background-repeat: repeat;
padding: 20px;
width: 240px;
height: 190px;


Tip: The border-radius property is actually a shorthand property for the border-top-left-radius, border-top-right-radius, border-bottom-right-radius and border-bottom-left-radius properties.

CSS border-radius - Specify Each Corner

The border-radius property can have from one to four values. Here are the rules:

Four values - border-radius: 15px 50px 30px 5px; (first value applies to top-left corner, second value applies to top-right corner, third value applies to bottom-right corner, and fourth value applies to bottom-left corner):

Four values - border-radius: 15px 50px 30px 5px:

The code for this:
border-radius: 15px 50px 30px 5px;

Three values - border-radius: 15px 50px 30px; (first value applies to top-left corner, second value applies to top-right and bottom-left corners, and third value applies to bottom-right corner):

Three values - border-radius: 15px 50px 30px:

The code for this:
border-radius: 15px 50px 30px;

Two values - border-radius: 15px 50px; (first value applies to top-left and bottom-right corners, and the second value applies to top-right and bottom-left corners):

Two values - border-radius: 15px 50px:

The code for this:
border-radius: 15px 50px;

One value - border-radius: 15px; (the value applies to all four corners, which are rounded equally:

One value - border-radius: 15px:

The code for this:
border-radius: 15px;

You could also create elliptical corners:

Ellipse border - border-radius: 50%:


The code for this:
border-radius: 50%;

The border-radius Property

Elliptical border - border-radius: 50px / 15px:

The code for this:
border-radius: 50px / 15px

Elliptical border - border-radius: 15px / 50px:

The code for this:
border-radius: 15px / 50px

Defines the shape of the border of the bottom-left corner
Add rounded borders to the bottom-left corner of two div-elements:

border-bottom-left-radius: 25px:

The border-bottom-left-radius property defines the radius of the bottom-left corner.

If two values are set; the first one is for the bottom border, the second one for the left border.

Закруглення кутів

Закруглення кутів за допомогою властивості 'border-radius'

Свойство позволяет закруглить углы строчных и блочных элементов. Кривая для каждого угла определяется с помощью одного или двух радиусов, определяющих его форму — круга или эллипса. Радиус распространяется на весь фон, даже если элемент не имеет границ, точное положение секущей определяется с помощью свойства background-clip.
Свойство border-radius позволяет закруглить все углы одновременно, а с помощью свойств:
border-top-left-radius, border-top-right-radius, border-bottom-right-radius, border-bottom-left-radius
можно закруглить каждый угол отдельно.

Значення, які задані через /, визначають горизонтальні та вертикальні радіуси. Ця властивість не наслідується.

Примеры:

При визначенні конкретного кута з використанням властивості 'border-radius' вказується цей конкретний кут шляхом вказання його положення:
border-top-left-radius, border-top-right-radius,
border-bottom-right-radius, border-bottom-left-radius
При цьому, для прикладу, якщо задати тільки два значення властивості 'border-radius' то перше значення закруглить верхній лівий та нижній правий кути, а друге — верхній правий та нижній лівий.

Ось приклади фигур із різними значеннями властивості 'border-radius':


r1: border-radius: 0 0 20px 20px;


r2: border-radius: 0 10px 20px;


r3: border-radius: 10px 20px;


r4: border-radius: 10px/20px;

r5: border-radius: 5px 10px 15px 30px/30px 15px 10px 5px;


r6: border-radius: 10px 20px 30px 40px/30px;



r7: border-radius: 50%;

r8: border-top: none; border-bottom: none; border-radius: 30px/90px;

r9: border-bottom-left-radius: 100px;

r10: border-radius: 0 100%;


r11: border-radius: 0 50% 50% 50%;

r12: border-top-left-radius: 100% 20px; border-bottom-right-radius: 100% 20px;






Тут створимо червоний квадрат, заокруглимо його кути та дамо границю у вигляді прошитих ниток
