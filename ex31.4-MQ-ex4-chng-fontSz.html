CSS Media Queries - change font size with MQ

In this example4, we can change the font size of an element on different screen sizes:

Resize the browser window to see the responsive effect!
On screens that are 992px wide or less, the columns will resize from four columns to two columns.
On screens that are 600px wide or less, the columns will stack on top of each other instead of next to eachother.


Example4 This is DIV-element.

/* Create div-element and style it */
div {
background-color: pink;
padding: 20px;
}
The code for media queries are:

/* If screen size is more than 600px wide, set the font-size of div-element to 80px */
@media screen and (min-width: 600px) {
div {
font-size: 80px;
}
}

/* If screen size is 600px wide, or less, set the font-size of div-element to 30px */
@media screen and (max-width: 600px) {
div {
font-size: 30px;
}
}
