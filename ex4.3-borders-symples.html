Як рамка, тут застосовано велике пропорційне єдине зображення без повторень і розтягувань середньої частини.
Через те, що 'border-width' не може бути вказаний в процентах, був застосований спосіб вирахування пропорцій сторін рамки шляхом обчислення значення властивості через 'calc'.

Для розрахунків беремо значення:


В результаті отримаємо:

для зовнішнього 'div':
position: relative;
padding-top: calc(2048*100%/1442);
background: url(http://4.bp.blogspot.com/-2EuZ69vi_-Y/UkEPKvIpW6I/AAAAAAAAECA/SpdlzrqQ_UU/s00/ramka.png) no-repeat;
background-size: 100% auto;

а для внутрішнього 'div':
position: absolute;
top: calc(510*100%/2048);
left: calc(155*100%/1442);
width: calc(100% - 155*2*100%/1442);
height: calc(100% - (760 + 510)*100%/2048);
overflow: auto;
