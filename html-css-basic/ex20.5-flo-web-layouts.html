Web Layout Example using float and claerfix hack

It is also common to do entire web layouts using the float property

Here the page is divided into 4 elements: title, menu, main information section and footer;
The menu and main information section put into div with the class clearfix:

.clearfix::after {
content: "";
clear: both;
display: table;
}

They are made in the form of 2 columns where the float-property was applied:

.column {
float: left;
padding: 15px;
}

.menu {
width: 25%;
}

.content {
width: 75%;
background-color: orange;
}
