Link as push brackets

This example demonstrates an example to display link with a brackets which change their position on hover: 

			a:link, a:visited {
			  color: white;
			  padding: 0 5px;
			  font-family: 'Poppins', sans-serif;
			  font-size: 20px;
			  font-weight: 700;
			  text-decoration: none;
			}
			
			a:hover.pushBrackets, 
			a:active.pushBrackets {
			  padding: 0;
			}

			a:hover.pushBrackets span, 
			a:active.pushBrackets span {
			  margin: 0 10px;
			  color: red;
			}

Here I added a class pushBrackets.
