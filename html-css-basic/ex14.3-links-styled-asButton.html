Link as Button

This example demonstrates a more advanced example where we it was combined several CSS properties to display link as box/button:

A link styled as a button:

			a:link, a:visited {
			  display: inline-block;
			  color: white;
			  padding: 14px 25px;
			  text-align: center;
			  font-family: 'Poppins', sans-serif;
			  font-size: 20px;
			  font-weight: 700;
			  text-decoration: none;
			  background-color: #f44336;
			  border-radius: 10px;
			}
			
			a:hover, a:active {
			  background-color: red;
			}
