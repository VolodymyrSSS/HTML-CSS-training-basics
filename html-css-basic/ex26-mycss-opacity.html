CSS Opacity / Transparency

The opacity property specifies the opacity/transparency of an element.

Transparent Image

The opacity property can take a value from 0.0 - 1.0. The lower value, the more transparent:

The code for this:
.opacity02>img {
opacity: 0.2;
filter: alpha(opacity=20); - for IE8 and earlier
}

.opacity05>img {
opacity: 0.5;
filter: alpha(opacity=50); - for IE8 and earlier
}

Image with 20% opacity:0.2

Image with 50% opacity:0.5

Image is not transparent, opacity:1 (default)

Note: IE8 and earlier use filter:alpha(opacity=x). The x can take a value from 0 - 100. A lower value makes the element more transparent.

Transparent Hover Effect

The opacity property is often used together with the :hover selector to change the opacity on mouse-over:

The code for this:
.op-05>img {
opacity: 0.5;
filter: alpha(opacity=50); - for IE8 and earlier
}

.op-05>img:hover {
opacity: 1.0;
filter: alpha(opacity=100); - for IE8 and earlier
}

Note: In IE, a !DOCTYPE must be added for the :hover selector to work on other elements than the a element.

An example of reversed hover effect

The code for this:
.op-03>img {
opacity: 1.0;
filter: alpha(opacity=100); - for IE8 and earlier
}

.op-03>img:hover {
opacity: 0.3;
filter: alpha(opacity=30); - for IE8 and earlier
}

Transparent Box

With opacity:

When using the opacity property to add transparency to the background of an element, all of its child elements inherit the same transparency. This can make the text inside a fully transparent element hard to read:

opacity 0.1

opacity 0.3

opacity 0.6

opacity 1 (default)

With RGBA color values:

If you do not want to apply opacity to child elements, like in our example above, use RGBA color values.
The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).
The following example sets the opacity for the background color and not the text:

10% opacity

30% opacity

60% opacity

default
