Методи форми 'POST' та 'GET'

з 'POST' на сторінку page1.html будуть перенаправлятись дані, які були введені в формі а в браузерному рядку буде мінятись тільки змінений адрес сторінки

- атрибут 'novalidate' - відправляє дані введені в форму без перевірки їх корректності
- атрибут 'novalidate' з 'on' - автоматично заповнить даними поле форми після її віправки
- атрибут 'enctype' з 'aplication/x-www-form-urlencoded' - якщо дані містять якісь напр кирилечні букви то вони будуть замінятись на їх 16-ні значення а пробіли на знаки '+' (це встановлено по-замовчуванню)
- атрибут 'enctype' з 'multipart/form-data' - дані не кодуються і використовуються для відправки інших файлів (наприклад зображень) з формою
- атрибут 'enctype' з 'text/plain' - дані не кодуються, а просто відправляються із заміною пробілів на знаки '+'

з 'GET' на сторінку page2.html будуть перенаправлятись дані, які були введені в формі а в браузерному рядку будуть відображатись на додаток до адресу сторінки ще й усі імена, які використовували в цій формі

- атрибут 'target' з наприклад 'iframe' - буде завантажувати результат в цей 'iframe'

Що таке 'input' ?
'input' - інтерактивний елемент управління, який використовується для управління формою


Текстове поле: for input type="text"
Поле-пароль: for input type="password"
Поле для e-mail: for input type="email"
Поле для URL: for input type="url"
Поле для пошуку: for input type="search"
Поле для телефону: for input type="tel"

Атрибути текстових інпутів

Текстове поле з максимальною кількістю введення символів в полі форми: for input type="text" autocomplete="on" maxlength="5"
Текстове поле з попереднім написом в полі форми: for input type="text" autocomplete="on" placeholder="Введіть ваше ім'я"
Текстове поле з обов'язковим введенням символів в поле форми (також забрати 'novalidate' якщо є): for input type="text" autocomplete="on" required
Текстове поле тільки для читання: for input type="text" autocomplete="on" readonly
Текстове поле з перевіркою орфографії введеного тексту: for input type="text" autocomplete="on" spellcheck="true"
Текстове поле з заданим шаблоном (тут для введення e-mail: тобто повинні бути введені символи як '@'' та '.'): for input type="text" autocomplete="on" pattern="email"
Текстове поле для обробки введених тих даних які вказані в імені.
Увага! імена можна і потрібно задавати для усіх типів інпутів!
here, we put the name and code for this is: input type="text" autocomplete="on" name="myName"

Атрибути кнопкових інпутів

Атрибут, який по-замовчуванню відмічає галочку в полі чекбоксу
Чекбокс4: for input type="checkbox" name="r1"
Чекбокс5: for input type="checkbox" name="r2" checked
Чекбокс6: for input type="checkbox" name="r3"

Атрибути поля-діапазона з визначеними границями цього діапазону та кроку зміни значень:
for input type="range" min="0" max="1000" step="20"

Атрибути для відправки файлів типу 'audio':
for input type="file" accept="audio/*"
Атрибути для відправки файлів типу 'audio':
for input type="file" accept="video/*"
Атрибути для відправки файлів типу 'image' з конкретними розширеннями: 'jpeg' та 'png'
for input type="file" accept="image/jpeg, image/png"
При цьому файли, які неможливо відправити будуть підсвічуватись сірим
Атрибут 'multiple' вказує на можливість декількох одночасних виборів for input type="file" multiple
