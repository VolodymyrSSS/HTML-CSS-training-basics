Responsive Images

Responsive images will automatically adjust to fit the size of the screen.

}


Center an Image

To center an image, set left and right margin to auto and make it into a block element:

The code for it is:
img {
display: block;
margin-left: auto;
margin-right: auto;
}

Transparent Image

The opacity property specifies the transparency of an element. The opacity property can take a value from 0.0 - 1.0. The lower value, the more transparent:

Image with 20% opacity:0.2

Image with 50% opacity:0.5

Image is not transparent, opacity:1 (default)

Note: IE8 and earlier use filter:alpha(opacity=x). The x can take a value from 0 - 100. A lower value makes the element more transparent.

The parts of the code for it are:

div {
width: 30%;
float: left;
text-align: center;
margin: 5px;
}
div.opacity02>img {
opacity: 0.2;
/* For IE8 and earlier */
filter: alpha(opacity=20);
}
div.opacity05>img {
opacity: 0.5;
/* For IE8 and earlier */
filter: alpha(opacity=50);
}
img {
width: 230%;
height: 160px;
}

display: block;
margin-left: auto;
margin-right: auto;
}

img:hover {
-webkit-transform: scaleX(-1);
transform: scaleX(-1);
}

Image Modal (Advanced)

Let's look at the example to demonstrate how CSS and JavaScript can work together.
First, use CSS to create a modal window (dialog box), and hide it by default.
Then, use a JavaScript to show the modal window and to display the image inside the modal, when a user clicks on the image:

×

In this example, we use CSS to create a modal (dialog box) that is hidden by default.

We use JavaScript to trigger the modal and to display the current image inside the modal when it is clicked on. Also note that we use the value from the image's "alt" attribute as an image caption text inside the modal.

The part of the code for it is:
div img {
display: block;
margin-left: auto;
margin-right: auto;
}

img:hover {
-webkit-transform: scaleX(-1);
transform: scaleX(-1);
}
