Завдання: створити дерево списку, гілками якого є вузли з іконками

Додатковими вимогами для верстки також є:
- Підтримка багаторядкових назв у вузлах;
- При наведенні на іконку, курсор повинен перетворитись на вказівний палець;

Для створення дерева надається макет-список ul/li та посилання на картинки для створення спрайта:

  • Розділ 1
    (містить два рядки)
    • розділ 1.1
    • Сторінка 1.2
      (містить два рядки)
  • Розділ 2
    (містить два рядки)

посилання на картинки для іконок вузлів надаються нижче:

Для вирішення задачі можна застосувати принцип двоколонкової верстки із застосуванням властивості float + margin. Де іконки будуть лівою колонкою а зміст гілок дерева списку - правою колонкою. Проте, це ускладнить вирішення задачі, тому можна усім іконкам задати float: left; а для контексту гілок дерева, зробити відступ звичайними margin.
