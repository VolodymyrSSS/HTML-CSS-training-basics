Use float with a list of hyperlinks to create a horizontal NAVIGATION MENU

Here a property of 'float: left;' was applied to each list-item

li {
float: left;
}

The each link has property as 'display: inline-block;' to ensure the inline-block behavior and not simple text-inline one

li a {
display: inline-block;
color: white;
text-align: center;
padding: 14px 16px;
text-decoration: none;
}
