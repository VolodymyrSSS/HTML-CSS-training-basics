Using CSS to style an HTML Form




Styling Input Fields

Use the 'width' property to determine the width of the input field.

An example of the code:
input {
width: 100%; -> A full-width input field
}

But this code will apply to all 'input' elements.

If you only want to style a specific input type, you can use attribute selectors:
input[type=text] -> will only select text fields
input[type=password] -> will only select password fields
input[type=number] -> will only select number fields
etc...

Padded Inputs

Use the padding property to add space inside the text field.
When you have many inputs after each other, you might also want to add some margin, to add more space outside of them.

The example of the code:
input[type=text] {
width: 100%;
padding: 12px 20px;
margin: 8px 0;
box-sizing: border-box;
}

Note that we have set the box-sizing property to border-box.
This makes sure that the padding and eventually borders are included in the total width and height of the elements.

Bordered Inputs

Use the border property to change the border size and color, and use the border-radius property to add rounded corners.

The example of the code:
input[type=text] {
border: 2px solid red;
border-radius: 4px;
}

If you only want a bottom border, use the border-bottom property, like in the field for input ID number of the form above:

The example of the code:
input[type=number] {
border: none;
border-bottom: 2px solid red;
}

Colored Inputs

Use the background-color property to add a background color to the input, and the color property to change the text color

The example of the code:
input[type=text] {
background-color: #3CBC8D;
color: white;
}

Or the code like in the field for input password of the form above:
input[type=password]:focus {
background-color: lightpink;
}

Focused Inputs

By default, some browsers will add a blue outline around the input when it gets focus (clicked on).
You can remove this behavior by adding outline: none;

Use the :focus selector to do something with the input field when it gets focus:

First example of the code:
input[type=text] {
width: 100%;
padding: 12px 20px;
margin: 8px 0;
box-sizing: border-box;
border: 1px solid #555;
outline: none;
}

input[type=text]:focus { -> to add a background color to the text field when it gets focused (clicked on)
background-color: lightblue;
}

Second example of the code:
input[type=text] {
width: 100%;
padding: 12px 20px;
margin: 8px 0;
box-sizing: border-box;
border: 3px solid #ccc;
-webkit-transition: 0.5s;
transition: 0.5s; -> to animate the border color (takes 0.5 seconds to change the color on focus)
outline: none;
}

input[type=text]:focus { -> to add a black border color to the text field when it gets focused (clicked on)
border: 3px solid #555;
}

Input with icon/image

If you want an icon inside the input, use the background-image property and position it with the background-position property.
Also notice that we add a large left padding to reserve the space of the icon

The example of the code:
input[type=text] {
width: 100%;
box-sizing: border-box;
border: 2px solid #ccc;
border-radius: 4px;
font-size: 16px;
background-color: white;
background-image: url('searchicon.png');
background-position: 10px 10px;
background-repeat: no-repeat;
padding: 12px 20px 12px 40px;
}

Animated Search Input

In this example we use the CSS transition property to animate the width of the search input when it gets focus.

The example of the code:
input[type=text] {
width: 130px; -> this is initial width of the field
box-sizing: border-box;
border: 2px solid #ccc;
border-radius: 4px;
font-size: 16px;
background-color: white;
background-image: url('searchicon.png');
background-position: 10px 10px;
background-repeat: no-repeat;
padding: 12px 20px 12px 40px;
-webkit-transition: width 0.4s ease-in-out;
transition: width 0.4s ease-in-out;
}

input[type=text]:focus {
width: 100%; -> this is width when it gets focus
}

Styling Textareas

The example of the code:
textarea {
width: 100%;
height: 150px;
padding: 12px 20px;
box-sizing: border-box;
border: 2px solid #ccc;
border-radius: 4px;
background-color: #f8f8f8;
resize: none;
}

Use the resize property to prevent textareas from being resized (disable the "grabber" in the bottom right corner).

Styling Select Menus

A styled select menu.

The example of the code:
select {
width: 100%;
padding: 16px 20px;
border: none;
border-radius: 4px;
background-color: #f1f1f1;
}

Styling Input Buttons

Tip: use width: 100% for full-width buttons

The example of the code:
input[type=button], input[type=submit], input[type=reset] {
background-color: #4CAF50;
border: none;
color: white;
padding: 16px 32px;
text-decoration: none;
margin: 4px 2px;
cursor: pointer;
}
