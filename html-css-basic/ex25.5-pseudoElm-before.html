CSS Pseudo-elements

The ::before Pseudo-element

A pseudo-class is used to define a special state of an element while pseudo-element is used to style specified parts of an element. For example, pseudo-element can be used to:
- Style the first letter, or line, of an element;
- Insert content before or after the content of an element;
Pseudo-classes can be combined with HTML classes. In this example, when you hover over any link, or use :hover pseudo-class on an 'a' elementit, it will appear the angle-sign '>' in front of the link:

Hover over every link to see the animation appearence of the '>'-sign.

            <div class="menu">
              <a href="#">Link 1</a>

.menu {
   display: flex;
   flex-direction: column;
   font-size: 2em;
}

.menu a {
   display: inline-block;
   width: 100px;
   text-decoration: none;
   color: brown;
   line-height: 1.5;
   transition: margin 0.2s ease-in-out;
}

.menu a:hover, .menu a:focus {
   position: relative;
   text-shadow: 0 0 5px red;
   margin-left: 0.5em;
   outline: none;
}

.menu a::before {
   display: inline-block;
   position: absolute;
   content: '>';
   left: -1em;
   color: transparent;
}
