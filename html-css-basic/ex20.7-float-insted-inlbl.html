float замість display:inline-block

За допомогою float можна розташовувати блочні елементи в рядку, подібно до дії властивості display: inline-block

.gallery li {
float: left;
list-style: none;
border: 1px solid black;
text-align: center;
margin: 5px;
}

Елементи з float:left рухаються вліво, а якщо це неможливо, то вниз, автоматично адаптуючись під ширину контейнера. Тут ми отримуємо ефект подібний до display: inline-block, але з особливостями float.
