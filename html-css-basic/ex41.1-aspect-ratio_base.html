CSS aspect-ratio Property

The aspect-ratio property allows you to define the ratio between width and height of an element.

If aspect-ratio and width properties are set, the height will follow in the defined aspect ratio.

RED BOX example

So, aspect ratio is width/height.

div {
   background-color: red;
   width: 300px;
   aspect-ratio: 3/2;
}

Aspect-ratio of 3/2 and width set to 300px makes height of red box 2/3 of width in this case.

quick RED CIRCLE

There are lots of ways to make circles in CSS, but this is definitely the most minimal - using aspect-ratio property:

div {
   inline-size: 30vmin;
   aspect-ratio: 1;
   border-radius: 50%;
   background-color: red;
}

Tip: Use the aspect-ratio property in responsive layouts where elements often vary in size and you want to preserve the ratio between width and height.
