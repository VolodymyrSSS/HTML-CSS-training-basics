CSS Multiple Columns

The CSS multi-column layout allows easy definition of multiple columns of text - just like in newspapers:

    The following lists all the multi-columns properties:
  1. column-count - Specifies the number of columns an element should be divided into
  2. column-fill - Specifies how to fill columns
  3. column-gap - Specifies the gap between the columns
  4. column-rule-style - Specifies the style of the rule between columns
  5. column-rule-width - Specifies the width of the rule between columns
  6. column-rule-color - Specifies the color of the rule between columns
  7. column-rule - A shorthand property for setting all the column-rule-* properties
  8. column-span - Specifies how many columns an element should span across
  9. column-width - Specifies a suggested, optimal width for the columns
  10. columns - A shorthand property for setting column-width and column-count
