CSS Media Queries - Min Width to Max Width

You can also use the (max-width: ..) and (min-width: ..) values to set a minimum width and a maximum width.

In this example6, we can change the appearance of DIV on different screen sizes.

Resize the browser window to see the responsive effect!
When the browser's width is between 600 and 900px, change the appearance of a div-element:


Example: this is the 1st DIV-element.

/* Create a div-element and style it: */
div {
font-size: 1.3em;
font-weight: 500;
font-style: italic;
background-color: pink;
}
The code for media queries are:
div {
@media screen and (max-width: 900px) and (min-width: 600px) {
div {
font-size: 50px;
padding: 50px;
border: 8px solid black;
background: yellow;
}
}

Using an additional value

We could add an additional media query to our already existing one using a comma (this will behave like an OR operator):

In the example below, when the browser's width is between 600 and 900px OR above 1100px, change the appearance of DIV.

Example: this is the 2nd DIV-element.

/* That is the same div-element: */
div {
font-size: 1.3em;
font-weight: 500;
font-style: italic;
background-color: pink;
}
The code of media queries for the 2nd div-element:
div {
@media screen and (max-width: 900px) and (min-width: 600px), (min-width: 1100px) {
div {
font-size: 50px;
padding: 50px;
border: 8px solid black;
background: yellow;
}
}
