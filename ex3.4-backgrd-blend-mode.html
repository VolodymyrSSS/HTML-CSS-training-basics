CSS background-blend-mode property

The background-blend-mode property defines the blending mode of each background layer (color and/or image).

Note: Edge/Internet Explorer do not support the background-blend-mode property.

Let's take the two images for background to be blended:



Specify the blending mode to be 'lighten':


The code for this:
width: 400px;
height: 400px;
background-repeat: no-repeat, repeat;
background-image: url("./img/img_tree.gif"), url("./img/paper.gif");
background-blend-mode: lighten;

Specify the blending mode to be 'darken':


The code for this:
width: 400px;
height: 400px;
